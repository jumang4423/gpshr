## usage

- ```gpshr``` 

    see help

- ```gpshr -install foo``` 

    install sound sctipts into foo directory

- ```gpshr -uninstall foo``` 

    uninstall sound script from foo

- ```gpshr -import foo``` 

    import sound from foo<!--
